レシート解析ツール 使い方ガイド

最終更新日: 2026/01/29

事前準備 初期設定 操作手順 補足

1. 事前準備

1-1. Gemini APIキーの取得

  1. 次のURLにアクセス: https://aistudio.google.com/api-keys
  2. 右上の「APIキーを作成」をクリック
  3. キー名をわかりやすく入力（プロジェクトはそのままでもOK）
  4. 作成したキーが一覧に表示されるのでクリック
  5. APIキーをコピー

1-2. APIキーの登録

  1. スプレッドシートのメニューから 設定 > APIキーの設定 をクリック
  2. 先ほどコピーしたキーを入力して保存

2. 初期設定（最初に1回）

2-1. 設定シート

  • 対象フォルダURL（スキャン対象のDriveフォルダURL）
  • 区切り文字（ファイル名生成時の区切り）
  • 解析上限（1回の解析件数）
  • 貸方勘定科目(クレカ) / 貸方勘定科目(それ以外)
  • 貸方補助科目(クレカ)：カード情報 / 空欄

2-2. ファイル名ルール

  • 項目名ごとに「順番」「使用可否」を設定
  • ファイル名の構成を自由にカスタマイズ可能

2-3. 移動先リスト

  • キーワードと移動先フォルダURLを登録
  • 解析シートの「移動先」列にキーワードを入れると移動できます

2-4. 勘定科目ルール

  • 仕訳ルールを記載（AIがルールに沿って判定）

3. 操作手順

3-1. レシート解析

  1. メニューから 1. レシート解析 を実行
  2. 解析結果が「解析シート」に反映されます
  3. 抜けや誤りがあれば編集してください
  4. 変更案を再生成（全行） を実行するとファイル名の変更案が更新されます
  5. 画像プレビューを開く を押すと、選択した行の画像が右側に表示されます
補足
・解析済みのファイルは再スキャン時にスキップされます。
・GASの制限（6分）対策として1回50件までの上限があります。
・50件ずつ繰り返せば全件解析できます。
・上限は設定シートで変更可能です。

3-2. ファイル名反映

メニュー 2. ファイル名反映 を実行すると、変更案の名前でDrive上のファイルが更新されます。

3-3. フォルダ移動

  1. 移動先リストにキーワードとフォルダURLを登録
  2. 解析シートの「移動先」列にキーワードを入力
  3. メニュー 3. フォルダ移動 を実行

※DriveのファイルIDは変わらないためリンクは壊れません。

3-4. マネフォ用解析

  1. 勘定科目ルールを記載
  2. 設定シートで貸方勘定科目/補助科目を設定
  3. メニュー 4. マネフォ用解析 を実行
  4. マネフォ用シートにデータが反映されます
  5. マネフォ用シートでも画像プレビューが有効です

3-5. マネフォCSVダウンロード

メニュー 5. マネフォCSVダウンロード を実行すると、マネーフォワードクラウド形式のCSVが出力されます。

4. 補足

4-1. モデルの変更

  • デフォルトは gemini-2.5-flash-lite
  • 精度に課題がある場合は gemini-3-flash-preview が推奨
  • ただし料金体系が変わる可能性があります
